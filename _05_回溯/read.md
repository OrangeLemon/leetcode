### 定义
递归函数下面的部分，就是回溯逻辑，回溯通常隐藏在递归下面。

### 解决问题范围
- 纯暴力搜索
- 组合
- 切割
- 子集
- 排列
- 棋盘
- N皇后

### 结构
多叉数，横行代表for循环，纵向代表递归

### 代码模式
```
void backTracking() {
    if(终止条件) {
        收集结果；
        return;
    }
    for(集合元素) {
        处理节点；
        递归函数；
        回溯操作；撤销处理节点，剪枝；
    }
    return;  
}

```
